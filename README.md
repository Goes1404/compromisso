## 🚀 Diferenciais Estratégicos

### 🎓 Jornada do Aluno (Estudante)
- **Central de Documentação**: Checklist inteligente e persistente para SiSU/ProUni com guia de organização em nuvem.
- **Simulador de Isenção Social**: Motor de cálculo de elegibilidade que atualiza o perfil do aluno automaticamente.
- **Sala de Aula Premium**: Interface industrial com player rastreável e suporte Aurora IA via Genkit.
- **Simulados Inteligentes**: Banco de questões randômico via RPC Supabase com feedback instantâneo.

### 👨‍🏫 Studio Master (Mentor)
- **Gestão de Trilhas 360**: Ferramentas de autoria para criação de módulos e anexo de múltiplos materiais.
- **Banco de Questões**: Repositório centralizado para alimentação de simulados em massa.
- **Console de Transmissão**: Estúdio Master para controle de status de sinal das reuniões externas.

### 🛡️ Gabinete de Gestão (Admin)
- **Auditoria de Documentos**: Visão térmica de quem está pronto para a matrícula e quem precisa de suporte.
- **Auditoria de Chats**: Supervisão ética e de segurança de todas as interações mentoria/aluno.
- **BI & Analytics**: Inteligência de dados sobre engajamento, evasão e performance da rede.

## 🎯 Configuração do Ecossistema (Supabase)

O arquivo `docs/database.sql` contém a estrutura definitiva. Ele configura:
1. **Tabelas Relacionais**: Perfis, Trilhas, Módulos, Conteúdos, Progressos e Logs.
2. **Funções RPC**: Motores de busca aleatória para simulados.
3. **Storage Policies**: Políticas de RLS para o bucket `avatars`.
4. **Triggers**: Criação automática de perfis no cadastro (SignUp).
